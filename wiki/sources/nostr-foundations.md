# Nostr Foundations for Entangle

The main Nostr building blocks relevant to Entangle are:

- NIP-01 for base event structure;
- NIP-17 and NIP-59 for private direct messaging patterns;
- NIP-90 as inspiration for request/result semantics;
- NIP-89 later for discovery-like surfaces if needed.

Entangle should not force itself into one existing NIP if the semantic model is richer. It should use Nostr as the cryptographic and transport substrate and define its own application protocol where needed.
