# Roadmap

## Roadmap Principles

The roadmap prioritizes trust before breadth. A platform for AI work must first prove that it can run, govern, observe, and audit a small number of important workflows reliably.

## MVP Roadmap

Timeframe: 3 months.

Deliverables:

- Workspace and authentication.
- Basic roles.
- Package registry v1.
- Graph composer v1.
- Graph validation.
- Model provider setup.
- Session launch.
- Sandboxed executor MVP.
- Artifact workspace v1.
- Approval gate v1.
- Session timeline.
- Audit log v1.
- OpenTelemetry baseline.
- CLI and API support for core flows.

Success criteria:

- First successful session under one hour for a new technical team.
- Complete trace and artifact evidence for every session.
- No unauthenticated mutating APIs.
- Tenant isolation tests pass.

## 3 Month Plan

Focus: private beta readiness.

Product:

- Improve onboarding templates.
- Add graph revision diff.
- Add package promotion.
- Add approval inbox.
- Add cost dashboard v1.
- Add search v1.

Integrations:

- GitHub.
- GitLab or Gitea.
- Slack.
- Webhooks.

Platform:

- Backup and restore drills.
- Staging environment.
- Load test baseline.
- Security review.

Customer:

- 3 to 5 design partners.
- Weekly feedback cycles.
- Support runbooks.

## 6 Month Plan

Focus: production launch.

Product:

- SAML and SCIM.
- Advanced RBAC and ABAC.
- Policy simulation and policy test suites.
- Trace replay.
- Artifact publication workflows.
- Analytics v2.
- Billing and entitlements.

Integrations:

- Jira.
- Linear.
- Teams.
- Google Drive.
- Confluence.

Platform:

- Production SLOs and alerting.
- Penetration test.
- SOC 2 Type I readiness.
- Self-host Helm chart beta.
- Data retention controls.

Customer:

- Paid pilot customers.
- Enterprise security review package.
- Customer success onboarding playbook.

## 12 Month Plan

Focus: scale and differentiation.

Product:

- Agent package marketplace or private catalog.
- Evaluation suites and regression testing.
- Advanced memory governance.
- Multi-graph workflows.
- Department-specific templates.
- Advanced cost optimization and model routing.

Integrations:

- ServiceNow.
- Zendesk.
- SharePoint.
- Snowflake or BigQuery export.
- SIEM export.

Platform:

- Multi-region deployment options.
- Enterprise self-host GA.
- Tamper-evident audit storage.
- Stronger sandbox tiers.
- Federation adapters including optional Nostr-compatible signed messaging.

Business:

- Enterprise plan mature.
- Partner ecosystem.
- Compliance roadmap toward SOC 2 Type II and ISO 27001.

## Future Expansion Opportunities

- AI operations benchmark suite.
- Policy pack marketplace.
- Regulated industry templates.
- Human workforce and AI workforce planning analytics.
- Agent reliability scoring.
- Automated incident response for agent failures.
- Cross-organization federation for controlled collaboration.
- Enterprise knowledge graph integration.

## Not Yet Prioritized

These ideas are intentionally deferred:

- Full consumer-facing no-code automation marketplace.
- Unrestricted autonomous agent swarms.
- Blockchain-based artifact notarization.
- Custom model training platform.
- Replacing customer systems of record.

Reason: they add complexity before the core governed-execution product is proven.
