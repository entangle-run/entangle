# Entangle Core

Entangle is a graph-native environment for modular AI organizations.

This means:

- the system is modeled as a graph, not as a hidden orchestration tree;
- the user is part of the graph;
- agents are explicit actors with identity and policy;
- work flows through a runtime subgraph during execution;
- messages and artifacts are distinct concerns;
- observability is a first-class property.

The key value proposition is not merely "decentralized AI." It is the explicit, governable decomposition of agent systems into interoperable actors and relations.
