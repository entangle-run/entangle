# Releases

This directory contains release-control material for Entangle product lines.

It is separate from `references/`: `references/` remains the canonical design
and specification corpus, while `releases/` tracks concrete release packets,
gate evidence, known limitations, and tag decisions.

## Product Lines

- [local](local/README.md) — Entangle Local release train.

Cloud and Enterprise release directories should be added only when the roadmap
allows work on those products.
