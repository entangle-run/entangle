# Releases

No public Entangle release packet is active yet.

The previous local release-train material was removed from the active
repository because it encoded an obsolete product framing. Entangle is a
federated runtime; same-machine operation is a development deployment adapter,
not a separate release line.

Future release packets should be organized by product capability or public
version, not by local-only profile. A release packet must include:

- supported deployment shape;
- verified smoke and test evidence;
- known limitations;
- upgrade and rollback notes;
- public documentation links;
- security and identity posture.
